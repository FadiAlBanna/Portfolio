## 🔧 How to Get Started

### 1. Development

```bash
cd "c:/Users/TOSHIBA/Desktop/Portfolio"
npm install
npm run dev
```

Opens at `http://localhost:5173/portfolio`

### 2. Customize Content

Edit JSON files in `src/data/`:

- Update `socialLinks.json` with your contact info
- Replace projects in `projects.json`
- Update skills, experience, certifications
- Add your resume PDF

### 3. Build

```bash
npm run build
```

### 4. Deploy

```bash
npm run deploy
```

Site will be live at `https://YOUR-USERNAME.github.io/portfolio`

## 📋 What Needs to Be Done

To make this fully yours:

1. **Update Social Links**
   - Edit `src/data/socialLinks.json`
   - Add your email, LinkedIn, GitHub, WhatsApp

2. **Add Your Projects**
   - Edit `src/data/projects.json`
   - Add project images to `src/assets/projects/`
   - Update descriptions and links

3. **Update Skills**
   - Modify `src/data/skills.json`
   - Add/remove skills relevant to you

4. **Add Work Experience**
   - Edit `src/data/experience.json`
   - Add your professional history

5. **Add Certifications**
   - Edit `src/data/certifications.json`
   - Upload PDFs/images to `src/assets/certificates/`

6. **Add Your Resume**
   - Place PDF in `src/assets/resume/`
   - Name it `CV-Fadi-AlBanna.pdf`

7. **Update About Section**
   - Edit text in `src/components/sections/AboutSection.jsx`
   - Update your professional biography

8. **Deploy to GitHub Pages**
   - Create GitHub repository
   - Run `npm run deploy`
   - Wait 1-2 minutes for activation
